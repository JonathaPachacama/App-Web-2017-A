Aplicaciones Web

Para entender el desarrollo web es imprescindible tener conocimiento de las 3 importantes tecnologías tratadas en este curso:

  • HTML: lenguaje de marcas empleado para la creación de documentos web.
  • CSS: lenguaje para definir las hojas de estilo que determinan la presentación de los documentos HTML.
  • JavaScript: lenguaje de programación utilizado para dotar de dinamismo a las páginas web. Imprescindible para sacar máximo provecho a las APIs de HTML5

Tecnologias Web con JavaSacript

Este curso esta enfocado en aprende angular.
Angularjs es un framework de aplicaciones web de código abierto mantenido principalmente por Google y por una comunidad de desarrolladores y empresas individuales. Su objetivo es simplificar el desarrollo y las pruebas de este tipo de aplicaciones, proporcionando un marco para el lado del cliente modelo-vista-controlador (MVC) y el modelo-vista-ViewModel (MVVM) arquitectura, junto con los componentes de uso común en aplicaciones dinámicas de Internet.

Gestion de TIC y unidades informaticas

Un tema importante que se tarat en este curso es COBIT que es un modelo para auditar la gestión y control de los sistemas de información y tecnología, orientado a todos los sectores de una organización, es decir, administradores IT, usuarios y por supuesto, los auditores involucrados en el proceso.
COBIT es un modelo de evaluación y monitoreo que enfatiza en el control de negocios y la seguridad IT y que abarca controles específicos de IT desde una perspectiva de negocios.
